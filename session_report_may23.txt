NEW — detect_eqh_eql()
  Finds Equal Highs / Equal Lows clusters (engineered liquidity pools).
  Real markets never make exact equal highs — second touch is a few pts
  higher/lower (e.g. 24850 → 24855). Two bugs fixed during build:

  Bug 1 FIXED — Tolerance too tight (0.02% = ±4.8 pts on NIFTY)
    Changed to 0.05% (±12 pts) so chained drift is caught correctly.

  Bug 2 FIXED — Clustering anchor wrong
    _cluster() was comparing each new point against group[0][0] (first
    point). Chained drift (24848 → 24851 → 24855) broke the cluster
    because pt3 was outside tolerance of pt1 even though it was within
    tolerance of pt2.
    Fix: changed group[0][0] → group[-1][0] (compare against previous
    point, not original anchor).

UPDATED — find_draw_on_liquidity()
  Now calls detect_eqh_eql() first. Prefers EQH/EQL cluster over
  single swing pivot when a cluster exists.
  Returns is_eqh_eql: bool and cluster_count: int.

NEW — _get_nse_h1_bias(fyers, symbol)
  Fetches 60-min candles (10 days) via Fyers.
  EMA(3) vs EMA(8):
    fast > slow * 1.0002  → BULLISH
    fast < slow * 0.9998  → BEARISH
    else                  → RANGING
  Fallback: RANGING when data unavailable (bot never crashes).

New package: nse_options/

  sensibull_client.py
    Wrapper around Sensibull public API (oxide.sensibull.com).
    Fetches option chains with Greeks. Retry with exponential backoff.
    No auth required (public analytics endpoint).

  option_cache.py
    TTL cache for API responses. Config loader from option_config.json.

  atm_strike_engine.py
    Strike gap mapping: NIFTY=50, FINNIFTY=50, MIDCPNIFTY=25,
    BANKNIFTY=100. Finds nearby strikes for a given spot price.

  greeks_engine.py
    Normalizes raw Sensibull chain into structured CE/PE Greeks dicts.

  option_pressure_engine.py
    Calculates PCR on OI and volume. Returns BULLISH/BEARISH/NEUTRAL
    bias based on put-call pressure.

  expiry_risk_engine.py
    Evaluates expiry risk (HIGH/MEDIUM/LOW) based on DTE.
    Bug FIXED: was using datetime.now().hour (local time) for the
    late-day expiry check. Fixed to datetime.now(IST).hour.

  option_chain_fetcher.py
    High-level wrapper. Fail-safe: returns neutral context if API fails.

  option_signal_context.py
    Main orchestrator. Enriches trade setups with options intelligence.
    If options bias contradicts direction: reduces risk_multiplier to 0.5.
    allow_strong_contradiction_block (default off) can block trade.
    Respects enabled: false — returns neutral context immediately.

  option_config.json
    Central config. Default: enabled=false, execution_allowed=false.
    Bot fully off by default. No options intelligence until enabled=true.
